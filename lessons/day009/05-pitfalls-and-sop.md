# ⑤ 避坑指南与实战 SOP

> 线性回归人人会跑，但这五个坑你不主动避，迟早会主动找上你。

## 坑 01：样本太少，贝塔不可信

- **现象**：拿最近 3 个月（约 60 天）数据估贝塔，今天算 0.8、下个月算 1.2，数字波动剧烈。
- **真相**：样本太小，标准误大，估出来的贝塔基本不可信；但窗口拉太长（>5 年）又会遇到「贝塔漂移」问题。
- **避坑动作**：起码用最近 2 年（500+ 个点）。经验法则：**2–3 年滚动窗口 + 每半年重估一次**。

## 坑 02：把贝塔当成铁打的常数

- **现象**：三年前算了一次 β，之后所有对冲、仓位计算都沿用这个数。
- **真相**：贝塔会随时间漂移。课程样本：工行 2018 年 β=0.6，2023 年 β=0.8，跟着银行业景气度浮动。
- **避坑动作**：做对冲 / 中性策略要**每月重估贝塔**。另外注意 Bloomberg 默认显示的 β 是 5 年月数据，慢得像化石，实战要用 1–2 年日数据。

## 坑 03：用物理学的 R² 标准苛求金融

- **现象**：看到回归 R²=0.05 就扔一边，觉得「这模型没用」。
- **真相**：物理学 R²=0.2 确实有点惨，但金融里股票日收益噪声极大——**R² 到 0.2 已经不错，0.5 算很强，0.8 基本是绑死了指数（像 ETF）**。学术论文里因子 R²=0.05 但 t-stat=4 也是有意义的（信号小但稳定）。
- **避坑动作**：用金融自己的标尺：R² > 0.6 强 / 0.3–0.6 中 / < 0.3 弱（但弱不代表没用，要看 t-stat）。

## 坑 04：数据脏——停牌 / 除权 / 涨跌停

- **现象**：回归结果莫名其妙，某几天把斜率拉得离谱。
- **真相**：停牌日的 0 收益、除权除息日的价格跳跃、涨跌停的收益截断，都会让回归失真。
- **避坑动作**：跑数据前必须做四件事——① 用复权价（后复权，Day 5 讲过）；② 删停牌日；③ 涨跌停日要么删要么 winsorize（缩尾处理）；④ 异常值检查 ±5σ。

## 坑 05：以为高贝塔就是好股票

- **现象**：牛市里专挑 β > 1.5 的票，觉得自己「眼光好」。
- **真相**：贝塔大只说明弹性大、放大效应强，牛市领涨熊市也领跌，**本身没有好坏之分**。混淆贝塔大小和股票质量是新手最常见错误。
- **避坑动作**：关键看你的风险偏好 + 当前市场阶段。低贝塔股票在熊市保护性强；高贝塔要等确认牛市后再上车。

## 线性回归实战 SOP

| # | 规则 | 一句话理由 |
| --- | --- | --- |
| 01 | 样本起点：2 年日数据（500+ 个点） | 低于此基本不可信 |
| 02 | 数据预处理：复权价 + 删停牌日 + winsorize 涨跌停日（去 ±5σ） | 脏数据进，垃圾结论出 |
| 03 | 用 ddof=1（贝塞尔修正）；statsmodels 的 OLS、scipy.stats.linregress 默认都对 | 别让自由度的小错误污染结果 |
| 04 | 看 t-stat 判断 α/β 显著性，t-stat > 2 才显著（p < 0.05） | 数字好看不代表统计上存在 |
| 05 | 贝塔每半年重估一次 | 跟踪漂移，别用化石参数 |
| 06 | 市场中性策略：多头总 β = 空头总 β，组合净 β ≈ 0 | 对冲要对称才有意义 |
| 07 | 金融数据 R² 标准：> 0.6 强 / 0.3–0.6 中 / < 0.3 弱（但可能信号显著） | 别拿物理标尺量金融 |

> [!IMPORTANT]
> **纪律 > 灵感**：量化做久的人都明白，赚钱的不是某一次绝妙判断，而是把规则坚持执行 1000 次。打印这一页，贴在你电脑边。
