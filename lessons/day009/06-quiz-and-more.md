**Q2. 你拿一只港股对沪深三百回归，得到 β=0.5、R²=0.1。这意味着什么？该不该这样建模？**

<details>
<summary>💡 答题思路提示（先自己想，再点开）</summary>

R²=0.1 意味着沪深三百只能解释这只港股 10% 的波动——这根「棍子」基本没贴住点。港股有自己的市场和汇率逻辑，用 A 股指数当自变量本身就值得怀疑。想一想：换成恒生指数做市场代理，β 和 R² 会变成什么样？

</details>

**Q3. 某基金报告年化阿尔法 8%，t-stat=1.5。你信不信他？为什么？**

<details>
<summary>💡 答题思路提示（先自己想，再点开）</summary>

回忆 SOP 第 4 条：t-stat > 2 才算统计显著（p < 0.05）。t-stat=1.5 意味着这个 8% 的阿尔法很可能是运气而不是能力——样本噪声完全可以产生这种数字。结论：不信，至少不能据此掏钱。

</details>

## 一页总结

1. **线性回归** = 在散点里找一根「最贴的棍子」；**最小二乘** = 让所有点到棍子的垂直距离平方和最小。
2. **为什么用平方**：① 数学上可写出闭式解一步算出；② 远点惩罚力度大（偏离 4 贡献 16），模型不会忽视极端值。
3. **贝塔（β）** = 市场敏感度，五种性格（0 / 0.7 / 1.0 / 1.4 / 1.9）对应不同弹性；**阿尔法（α）** = 扣掉市场后的真本事，能稳定做出年化 5% 就是业内顶级。
4. **R²** = 「解释百分比」，金融里 0.3–0.5 算正常，> 0.7 接近 ETF。β 说方向多陡，R² 说棍子贴多紧。
5. **残差** = 实际值 − 预测值，是统计套利策略最朴素的信号源：连续同号残差 = 超额收益在持续。
6. **市场中性策略** = 用 β 对冲市场风险（多头总 β = 空头总 β），只留阿尔法——私募经典打法。
7. 实测胜于教科书：工行实测 β=0.157（国家队维稳脱钩），隆基 β=1.44 但 R²=0.40（60% 是自己的故事）。
8. 线性回归是入门工具，真实金融关系常常非线性，后面要补决策树 / 集成学习。

## 延伸阅读

- Sharpe《Capital Asset Prices》（1964, Journal of Finance）— CAPM 原文，β/α 在金融里的源头
- Greene《Econometric Analysis》（第 8 版）— 计量经济学圣经，OLS / 多元回归 / 异方差 / 内生性最权威教材
- Tsay《Analysis of Financial Time Series》（第 3 版）— 金融时序经典，贝塔估计 / 因子模型 / GARCH 一站式
- Fama & French《The Cross-Section of Expected Stock Returns》（1992）— 三因子模型，贝塔的扩展
- James、Witten、Hastie《An Introduction to Statistical Learning》第 3 章 — 线性回归的数学和代码
- statsmodels OLS 文档 — Python 量化最常用的回归库，带 t-stat / p-value / 残差诊断
- Bridgewater 早期论文《How the Machine Works》— Dalio 团队怎么用回归做宏观因子分解
