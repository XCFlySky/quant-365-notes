# Day 009 · 线性回归与最小二乘

> 在一堆乱糟糟的散点里画一根「最贴的棍子」——这根棍子的斜率叫贝塔，截距叫阿尔法，它是整个量化金融最基础的工具，没有之一。

- 📂 阶段：P1 · 量化基础
- ⏱️ 时长：约 18 分钟
- 🎯 关键词：散点图直觉、最小二乘（OLS）、贝塔/阿尔法/R²、残差选股信号、市场中性

## 这节课为什么重要

把工商银行每天的涨跌画在竖轴，沪深三百当天的涨跌画在横轴，三年下来 700 多个点散在纸上，你画一根最贴的棍子穿过去——这根棍子的**斜率**就是大名鼎鼎的「贝塔」，告诉你这只股票跟市场跑得有多紧；**截距**叫「阿尔法」，告诉你这只股票自己有多大本事；**决定系数 R²** 告诉你这根棍子能解释多少波动。

这节课用三只性格完全不同的标的——工商银行（低贝塔）、隆基绿能（中高贝塔）、英伟达（高贝塔）——一次把 OLS / α / β / R² / 残差 / 市场中性策略全部讲透。学完你能算出自家股票的贝塔，看清它是个跟市场跑的乖孩子，还是有自己脾气的特立独行者。

## 学习目标

- [x] 理解散点图 +「最贴的线」的直觉，把抽象的「回归」落地为画图问题
- [x] 看懂最小二乘（OLS）为什么用平方而不是绝对值，以及它的两个数学好处
- [x] 掌握贝塔（β）、阿尔法（α）、决定系数（R²）在金融里的精确含义和典型量级
- [x] 学会用残差识别超额收益信号——整个统计套利策略的起点
- [x] 知道线性回归的四个常见坑（样本太少 / 贝塔漂移 / R² 标准 / 数据脏）
- [x] 理解市场中性策略 = 用 β 对冲市场风险、只留 α——私募里的核心打法

## 一点历史八卦

- **1809 年**，Gauss 用最小二乘法算行星轨道，奠定数学基础；
- **1885 年**，Francis Galton 研究「父亲身高 vs 儿子身高」时发现高个子父亲的儿子平均会向整体均值「回归」——他造了 regression 这个词，本意是「回到均值」；
- **1964 年**，William Sharpe 提出 CAPM，核心就是把单只股票收益对市场收益做线性回归，斜率叫贝塔、截距叫阿尔法。Sharpe 因此在 1990 年和 Markowitz 一起拿了诺贝尔奖；
- **1992 年**，Fama-French 三因子模型把贝塔扩展到三个维度——这一切的源头，都是这条最朴素的「最贴的线」。

> [!TIP]
> 本节最重要的两句话：**贝塔是市场敏感度，阿尔法是真本事；残差是量化选股最朴素也最强的信号源。**
