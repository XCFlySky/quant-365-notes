# ① R² 的直觉与公式

> β 是拟合线的「斜率」，R² 是拟合线的「贴合度」——一个说方向，一个说靠不靠谱，两个必须配套看。

## 几何直觉：棍子贴散点贴得有多紧

回忆 Day 9：用最小二乘法给「个股收益 vs 大盘收益」的散点图拟合一条棍子，棍子的斜率就是 β。今天讲第二个量：**这条棍子贴得有多紧**。

做法很直观：量每个散点到棍子的垂直距离（叫**残差**），所有残差越小，棍子贴得越紧，R² 越高。

| R² 取值 | 含义 |
| --- | --- |
| R² = 1 | 所有点都在棍子上，完美预测（现实中几乎不可能） |
| R² = 0 | 棍子还不如直接用均值猜，模型没解释任何东西 |
| R² = 0.85 | 模型解释了 85% 的波动，剩 15% 是无法解释的「噪声」（个股特性、黑天鹅、测量误差） |

```text
R² ∈ [0, 1]  越接近 1 → 棍子贴得越紧 → 模型解释力越强
```

## 金融里的真实刻度（课程样本）

| 标的 | 对沪深 300 的 R² | 解读 |
| --- | --- | --- |
| 沪深 300 ETF | 接近 1 | 几乎完美跟踪 |
| 平安银行 | 约 0.6 | 银行整体跟金融板块走 |
| 紫金矿业 | 约 0.3 | 它跟铜价、金价更相关 |
| 长江电力 | 约 0.2 | 防御股，自己一个节奏 |

还有一个有意思的统计：把沪深 300 成分股（300 只）分别对指数做回归，**R² 中位数约 0.6**——平均一只股票 60% 的波动能用大盘解释，40% 是它自己的故事。这也是为什么「买一只 ETF 就万事大吉」的想法有漏洞：跟大盘相关的部分可以一起跌。

## 公式三步走：R² = 1 − 残差方差 / 总方差

```text
第一步：SS_tot = Σ(y − ȳ)²        # 数据本身的总散开程度
第二步：SS_res = Σ(y − ŷ)²        # 模型预测后还剩多少没解释
第三步：R² = 1 − SS_res / SS_tot  # 模型解释掉的占比
```

白话：**R² 就是「模型解释掉的波动」占「全部波动」的比例。** 等价地，在一元回归里 R² 也等于「预测值与实际值相关系数的平方」。

## 三种股票画像：β 和 R² 配套看

这是这节课最实用的一张表（课程实测数据，三年日频）：

| 画像 | 特征 | 代表（课程样本） | 实战含义 |
| --- | --- | --- | --- |
| 中 β + 中 R² | 跟随大盘 | 平安银行 β=0.753 / R²=0.411；招商银行 β=0.673 / R²=0.314 | 大盘权重板块共振 |
| 高 β + 低 R² | 看似跟盘，实则独立 | 紫金矿业 β=0.986 / R²=0.240 | β 会骗人，76% 是商品周期逻辑 |
| 低 β + 低 R² | 防御独立 | 长江电力 β=0.123 / R²=0.022 | 几乎不跟大盘，自己的节奏 |

> [!WARNING]
> **高 β 低 R² 陷阱**：紫金矿业 β ≈ 1，看起来像典型大盘股，但 R² 只有 0.24——大盘只能解释它 24% 的涨跌，剩下 76% 是铜价、金价、矿价这些独立逻辑。教科书直觉「β≈1 = 跟大盘紧贴」是错的。**β 告诉你平均斜率，R² 才告诉你散点贴不贴线。**

## 常见误读速查

| 说法 | 对不对 | 真相 |
| --- | --- | --- |
| 「R² 高 = 模型能赚钱」 | ❌ | R² 是解释力，不是盈利能力 |
| 「R² 低 = 模型没用」 | ❌ | 择时模型 R² 0.05~0.10 已属实战可用 |
| 「β 和 R² 看一个就行」 | ❌ | 一个管方向，一个管贴合，必须配套 |

> [!TIP]
> 一句话带走：**β 是「平均往哪走」，R² 是「走得有多齐」。** 体检一只股票，两个数字都要查。
