# Day 014 · R² 与策略评估

> Day 9 的 β 告诉你「股票跟大盘走的方向有多陡」，R² 告诉你「这条棍子贴得多紧」——两个值合起来才是完整画像。

- 📂 阶段：P1 · 量化基础
- ⏱️ 时长：约 16 分钟
- 🎯 关键词：拟合度、样本外 R²、解释 vs 预测

## 这节课为什么重要

R² 是评价模型解释力的量化指标。这节课讲清五件事：

1. **R² 的几何直觉**——在散点图里，它就是拟合线贴近数据点的程度；
2. **公式 1 − SS_res/SS_tot** 怎么算、物理含义是什么；
3. **高 R² 不一定是好事**——主动基金 R² > 0.95 就是 closet indexing（抱指数躺平）；
4. **Adjusted R²** 防止你靠加垃圾变量提分；
5. **样本内 R² vs 样本外 R²** 是检测过拟合最直接的工具，落差大于一半视为警告。

学完你能用三个数字（β + R² + 调整 R²）给任何一只股票或任何一个策略打出体检报告，看出它是真 alpha 还是假独立、是真本事还是抱指数。

## 学习目标

- [x] 理解 R² 的几何直觉——棍子贴散点的紧密度，以及 R² 在 0 到 1 区间的含义
- [x] 掌握 R² = 1 − SS_res/SS_tot 的计算和「解释方差占比」的核心解读
- [x] 区分「高 β + 高 R²」（跟随）、「高 β + 低 R²」（独立）、「低 β + 低 R²」（防御）三种股票画像
- [x] 知道为什么 Adjusted R² 比普通 R² 更可靠——加垃圾变量也能把 R² 推上去
- [x] 掌握样本内 vs 样本外 R² 双值报告纪律——样本外 R² 跌幅大于一半视为过拟合警告
- [x] 识别「主动基金 R² > 0.95」是 closet indexing 抱指数，不是真 alpha

## 一点历史八卦

- **1903 年**，Karl Pearson 提出相关系数，其平方就是 R² 的雏形；
- **2009 年**，耶鲁大学的 Martijn Cremers 和 Antti Petajisto 发表《How Active Is Your Fund Manager?》，提出 Active Share 指标，查了 1980–2003 年所有美国主动股票基金，发现**三成以上 R² > 0.95**——收着主动管理费，实际抱住指数躺平。论文一发，SEC 跟进监管，数千只基金被迫调整持仓或降费；
- **2020–2021 年的中国**上演过一模一样的剧情：千亿级「主动」基金清一色重仓茅台、宁德时代这一篮子「茅指数」「宁组合」，跟沪深 300 的 R² 在 0.85 以上。2022 年指数回调 35%，这些基金净值同步腰斩，投资者才发现主动费全白付了；
- **Fama & French** 的三因子模型对个股 R² 平均 0.7~0.9，是因子模型解释力的标杆数据。

> [!TIP]
> R² 这个看起来很学术的指标，在现实里是**揭穿伪装的探照灯**。散户买基金第一件事：看它跟指数的 R² 是多少。
