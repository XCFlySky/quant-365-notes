# Day 006 · 复利与贴现

> 金融世界里最基础的两个问题：「今天的钱，未来值多少」和「未来的钱，今天值多少」——这节课把这两个问题彻底讲透。

- 📂 阶段：P1 · 量化基础
- ⏱️ 时长：约 16 分钟
- 🎯 关键词：复利公式、72 法则、贴现现金流、NPV、IRR、三大复利杀手

## 这节课为什么重要

爱因斯坦把复利称作「第八大奇迹」，巴菲特一辈子财富的 89% 是 60 岁后赚到的——不是他变聪明，是前 60 年的复利在最后阶段集中显现。这节课讲四个核心工具：

1. **复利公式 FV=PV×(1+r)^N** 和 **72 法则**（5 秒心算翻倍年限）；
2. **贴现**（复利的逆运算）与 **NPV**（净现值）；
3. **IRR**（内部收益率）——识破一切金融产品话术的 X 光机；
4. **税、通胀、手续费**三大复利杀手。

学完你能自己算房贷、年金保险、教育金、银行理财的真实回报，识破销售员「年化 4.5% 二十年翻一倍」的话术——大多数这类产品的实际 IRR 只有 1.8–2.5%，远低于同期国债。

## 学习目标

- [x] 推出复利公式 FV = PV × (1+r)^N，理解为什么时间是复利里第一变量
- [x] 用 72 法则 5 秒钟心算「年化 r% 翻倍要多少年」
- [x] 掌握贴现 = 复利的逆运算，理解 NPV 和 IRR 的物理意义，而不只是 Excel 公式
- [x] 看懂年金保险 / 教育金 / 银行理财的「真实年化」，识破「年化 4.5%」销售话术
- [x] 量化通胀、税、手续费三大复利杀手对长期收益的吃损，以及为什么 Bogle 力推低费率指数基金

## 一点历史八卦

- 相传 **1955 年**有人问爱因斯坦人类最伟大的发明是什么，他的答案不是火、不是电、也不是核能，而是——**复利**。「懂复利的人就赚利息，不懂复利的人就付利息。」
- **巴菲特**一辈子的财富里，89% 是 60 岁之后赚到的。这不是因为 60 岁后突然变聪明，而是前 60 年的复利在最后阶段才真正显现威力。
- **2010 年代**国内某大型保险公司热销的「分红型年金保险」，销售员承诺「年化 4.5%、保本保收益、二十年翻一倍多」。可一旦把交保费的现金流序列拉出来用 IRR 反算，大多数实际收益只有 1.8%–2.5%，远低于同期国债。一个不会算 IRR 的家庭，就这样被困在二十年低收益合同里。
- Vanguard 创始人 **John Bogle** 把「低费率 + 复利」做成全民武器，他的名言是：「在长期投资里，1% 的费率差，等于偷走 1/3 的最终财富。」

> [!TIP]
> 本节最重要的两句话：**复利是时间的朋友，但只忠于懂它的人；学会算 IRR，你就有了识破金融产品话术的 X 光机。**
