# ⑤ 避坑指南与实战规则

> 每个市场都有自己的「死法」，逐个排雷。

## 五个常见的坑

### ⚠️ 坑 1：A 股 T+1 + 个股做空难

**现象**：想在 A 股做纯量化做空个股。
**真相**：T+1（今天买明天才能卖）+ 融券池小、利率 8–12%，个股做空几乎不可行。
**避坑动作**：用股指期货/ETF 做空整体头寸，个股只做多。

### ⚠️ 坑 2：港股流动性两极分化

**现象**：看到「便宜」的小盘港股就买。
**真相**：头部 50 只（腾讯/汇丰/美团）流动性极好，1000 名以后可能整天 0 成交——买得进卖不出。
**避坑动作**：量化选股必须加**流动性过滤**这一层。

### ⚠️ 坑 3：美股交易时间差

**现象**：美股开盘是北京时间 21:30（夏令时）/22:30（冬令时），熬夜盯盘。
**避坑动作**：用算法单 + 限价单 + 自动化，别拿睡眠换收益。

### ⚠️ 坑 4：加密合规风险

**真相**：中国大陆禁止加密交易，境外开户有平台跑路、银行卡冻结、汇兑限制等多重风险。
**避坑动作**：要做也小金额、主流所、不超过总资金 5%，合规自己评估。

### ⚠️ 坑 5：期货保证金爆仓

**真相**：新手在期货市场最常见的死法 = 仓位过重 + 隔夜跳空。
**避坑动作**：**单笔风险 ≤ 1% 总资金，永远不在收盘前 30 分钟加仓。**

## 市场选择 SOP

| # | 规则 |
| --- | --- |
| 01 | 0–3 个月：只做 A 股（免费数据 + 量化主战场 + 不熬夜） |
| 02 | 3–6 个月：加港股 ETF（2800/2828）做对冲和多空验证 |
| 03 | 6–12 个月：开美股账户，用 QQQ/SPY/IWM 做 ETF 因子组合 |
| 04 | 12+ 个月：可加国内期货期权（50ETF 期权、商品期货门槛低） |
| 05 | 加密：合规清楚 + 资金限额（≤5%）+ 只用 Binance/OKX 主流所 |
| 06 | 永远不做：外汇杠杆、欧洲股票（信息环境差，性价比低） |
| 07 | **至少 3 个市场的认知，1–2 个市场的实操深度——深度 > 广度** |

> [!IMPORTANT]
> 1–2 个市场的真功夫，胜过 5 个市场的浅尝辄止。
