> 股票/债券/期货/期权/外汇/加密，一节课过完所有能交易的市场——看清哪些适合你，哪些是披着机会外衣的坑。

- 📂 阶段：P1 · 量化基础
- ⏱️ 时长：约 22 分钟
- 🎯 关键词：股票/债券/期货、期权/外汇/加密、T+0 vs T+1

## 这节课为什么重要

量化要能**跨市场思考**。不同市场的规则（结算、做空、杠杆、交易时间）直接决定你的策略能不能落地。而且对中国大陆居民来说，每个市场的「进入通道」都不一样——QDII、港美券商、50 万验资门槛、加密合规风险，这节课全部讲清楚。

> [!NOTE]
> 核心结论先行：**不是所有资产都适合量化，也不是所有量化都需要去最复杂的市场。** 散户的实际可做集合 = A 股 + 港股 + 美股 + 国内期货期权 +（合规自担的）加密。

## 学习目标

- [x] 区分六大资产类的本质：股票/债券/期货/期权/外汇/加密
- [x] 知道每类资产的交易时间、结算规则、量化适合度
- [x] 看懂跨市场套利的逻辑（AH 折溢价、ETF 折价、期现）
- [x] 评估自己的资金能进哪些市场
- [x] 选定 1–2 个主战场，而不是什么都做一遍

## 散户的真实通道

- **A 股**：门槛最低，数据免费，但 T+1 + 个股做空难；
- **港股**：港美券商（富途/老虎/盈透）开户即可，T+0 + 双向做空；
- **美股**：同上，但有每年 5 万美元外汇额度 + 深夜交易时间；
- **国内期货期权**：商品期货门槛低，股指/期权需 50 万验资；
- **加密**：大陆开户处于灰色地带，合规与资金安全风险自担。
