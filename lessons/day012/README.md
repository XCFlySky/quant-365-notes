# Day 012 · 中心极限定理在金融

> 随便什么歪瓜裂枣的分布，只要样本够多，它们的「平均值」都会乖乖长成一口钟——这就是统计学的皇冠明珠。

- 📂 阶段：P1 · 量化基础
- ⏱️ 时长：约 18 分钟
- 🎯 关键词：CLT 直觉、Bootstrap 抽样、聚合后的分布

## 这节课为什么重要

中心极限定理（CLT）回答了一个看似矛盾的问题：**原始数据明明不服从正态分布，为什么统计工具（置信区间、t 检验、Sharpe ratio）还老是假设正态？** 答案是——我们不关心原始数据，我们关心的是「样本均值」，而样本均值的分布会逼近正态。

这节课跟你接下来 3 个月的学习关系很大：

1. 你会明白为什么 Bootstrap 重采样这种「无中生有」的方法居然有效；
2. 你会知道金融数据什么时候满足 CLT 的假设、什么时候偷偷违反（自相关、波动聚集）；
3. 你会学会一个实战魔法——**分组聚合**：组合收益比个股更正态，统计检验更可靠。

## 学习目标

- [x] 理解中心极限定理的直觉：大量独立样本均值会趋近正态分布
- [x] 通过可视化实验，亲眼见证 CLT 如何从任意分布收敛到正态
- [x] 掌握 Bootstrap 抽样法：用重采样估计置信区间
- [x] 理解金融数据何时满足、何时违反 CLT 假设（独立同分布）
- [x] 学会用分组聚合改善正态性，让统计工具更可靠

## 一点历史八卦

- **1733 年**，法国数学家 Abraham de Moivre 在《机遇原理》里第一次描述：多次抛硬币，正面次数的分布会形成钟形曲线。当时只被当成一个数学好奇心，没人想到它两百年后会成为现代统计学的基石；
- **1920 年代**，George Pólya 正式把它命名为「中心极限定理」；
- **1952 年**，Harry Markowitz 的现代组合理论直接假设收益服从正态分布——背后撑腰的就是 CLT；
- **2008 年金融危机后**，Nassim Taleb 等人猛烈抨击正态假设，指出金融收益的「肥尾」让传统 VaR 模型失效。争论至今未停，但 CLT 仍是量化金融的第一工具。

> [!TIP]
> 记住这节课的主线：**CLT 是工具，不是真理。** 它能用，但你要知道它的边界在哪。
