# ⑤ 避坑指南与实战规则

> 这些坑你不主动避，迟早会主动找上你。每个坑都附避坑动作，不只是描述现象。

## 坑 1：把价格自相关当成有意义

- **现象**：画出价格 ACF，看到 lag=1 到 lag=20 全是 0.95–1.00，兴奋地说「这股票可预测性极强」。
- **真相**：价格序列本身是非平稳的累加过程，自相关接近 1 是**数学必然**，不是信号（回忆 [Day 015](lessons/day015/01-stationarity.md)：价格几乎都不平稳）。
- **避坑动作**：有意义的永远是**收益（差分后）**的自相关，不是价格的自相关。分析前先把价格转成收益。

## 坑 2：看到弱自相关就以为可以预测

- **现象**：测出 ρ(r, 1) = 0.05，t 检验显著，立刻写策略上线。
- **真相**：统计意义 ≠ 经济意义。样本 n = 2500 时 0.05 都能显著，但 0.05 的自相关意味着 R² = 0.0025——扣掉手续费、滑点、税之后基本不剩。
- **避坑动作**：至少 ρ > 0.10 + 信号稳定 + 样本大，三者齐了才考虑做成策略。

## 坑 3：滞后阶数选错（要看 PACF）

- **现象**：给 AR(p) 模型拍脑袋选阶数，或者盯着 ACF 找 p。
- **真相**：ACF 是总相关，看不出 p；**PACF 在哪一阶截断（掉到不显著），那一阶就是 p**——这是标准方法。
- **避坑动作**：PACF 通常在 lag=1 到 lag=3 截断，大多数金融序列 AR(1) 或 AR(2) 已经足够，别堆高阶。

## 坑 4：不分时间段直接套教科书结论

- **现象**：教科书说「短期反转」，于是 2020 年代在美股做月度反转策略。
- **真相**：这是本课最大的教训。教科书结论建立在 1990 年代美股数据上，近十年实测标普月度自相关 +0.10，是**动量**——市场结构变了，结论失效。不验证就上线 = 历史套利。
- **避坑动作**：任何动量 / 反转 / 趋势策略，必须用最近 3–5 年实测验证 Hurst + 月度 ρ + Ljung-Box 三件套。

## 坑 5：把波动率自相关当成均值可预测

- **现象**：看到 |收益| 自相关 0.3，以为「可以预测明天涨跌」。
- **真相**：0.3 是**波动率**自相关——只能预测明天「波动幅度大不大」，预测不了「涨还是跌」。方向和波动是两个独立的维度。
- **避坑动作**：波动率策略和方向策略分开做。波动率聚集对应的打法是卖跨式期权、波动率目标仓位、跟踪 VIX——全都跟方向无关。

## 实战 SOP：自相关与预测

| # | 规则 |
| --- | --- |
| 01 | 做时间序列必看 ACF + PACF 两张图：先 ACF 看总记忆，PACF 选 AR 阶数 |
| 02 | Ljung-Box 检验 p < 0.05 才进下一步建模，接近白噪声不浪费时间 |
| 03 | 做收益预测前先看 \|收益\| 自相关——几乎所有金融资产都聚集 |
| 04 | 动量 / 反转策略必须实测最近 3–5 年 Hurst + 月度 ρ，教科书结论必验证 |
| 05 | Hurst > 0.55 趋势可用 / < 0.45 反转可用 / 0.45–0.55 随机别折腾 |
| 06 | 波动率策略和方向策略分开做——波动率聚集只指示波动率，不指示方向 |
| 07 | 策略上线后每季度滚动重测 Hurst + 自相关，关系反转就出场 |
