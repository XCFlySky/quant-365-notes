# Day 016 · 自相关与预测

> 收益本身几乎不可预测，但「波动有多大」高度可预测——这节课教你用一组指标判断任何标的是动量股还是反转股，以及波动率有没有记忆。

- 📂 阶段：P1 · 量化基础
- ⏱️ 时长：约 18 分钟
- 🎯 关键词：ACF/PACF、白噪声、Ljung-Box 检验、波动率聚集、Hurst 指数

## 这节课为什么重要

自相关是时间序列分析里最反直觉、最实用的工具之一。上一课（[Day 015 · 时间序列与平稳性](lessons/day015/README.md)）解决了「数据能不能用」，这节课解决「数据里有没有信号」。

讲清楚四件事：

1. **自相关的定义**——一个序列跟自己「滞后版本」的相关性，以及 ACF / PACF 两个工具；
2. **Ljung-Box 检验**——判断收益是不是真的「白噪声」（无法预测）；
3. **收益自相关 ≈ 0，但 |收益| 自相关 > 0**——这就是著名的「波动率聚集」现象，日度风控的核心；
4. **Hurst 指数**——一个数字区分趋势（> 0.5）、随机（= 0.5）、反转（< 0.5）三种特性。

今天用 SPY、QQQ、黄金期货、美元指数四个标的实测十年数据——结果会推翻几个教科书直觉：近十年标普500月度其实是**动量**不是反转，黄金和美元的 Hurst 都小于 0.5 呈**反转**特性。

## 学习目标

- [x] 理解自相关的定义：序列今天的值和昨天 / 前天 / N 天前的值的相关性
- [x] 区分 ACF（自相关函数）和 PACF（偏自相关函数）：一个看总相关，一个去除中间环节
- [x] 会用 Ljung-Box 检验判断序列是否是「白噪声」（无可预测信息）
- [x] 理解金融市场的反直觉现象：收益本身近似不可预测，但 |收益| 高度可预测（波动率聚集）
- [x] 用 Hurst 指数区分三种特性：趋势（动量可用）/ 随机游走 / 反转（短反可用）
- [x] 能用实测数据辨别教科书直觉对错：标普近 10 年是月度动量，黄金 / 美元 Hurst 呈反转

## 一点历史八卦

- **1900 年**，法国数学家 Louis Bachelier 在博士论文《投机理论》里首次提出金融价格是随机游走——比 Fama 的有效市场假说早了半个多世纪。当时被学界忽视（「金融不是严肃数学」），五十年后被翻出来，成为现代金融数学的起点；
- **1963 年**，分形几何之父 Benoit Mandelbrot 发现：收益本身不可预测，但收益的绝对值高度可预测——「波动率聚集」首次被记录；
- **1982 / 1986 年**，Engle 提出 ARCH 模型把波动率聚集数学化，学生 Bollerslev 推广成 GARCH——2003 年诺贝尔经济学奖。GARCH 至今是期权定价、风险管理、动态对冲的地基；
- **1951 年**，英国水利工程师 Harold Hurst 研究尼罗河水位时提出 Hurst 指数，后来被 Mandelbrot 引入金融，衡量序列的「长期记忆」。

> [!TIP]
> 这节课最深的教训：**市场结构在变，真实数据每年都在颠覆「应该」**。任何动量 / 反转 / 趋势策略，必须用最新数据实测验证，不能套用教科书结论。
