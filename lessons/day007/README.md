# Day 007 · 风险与方差

> 金融里的「风险」不是「会不会亏」，而是「收益散得多开」——这节课用一个数字（波动率）给所有资产排好「游戏难度」。

- 📂 阶段：P1 · 量化基础
- ⏱️ 时长：约 18 分钟
- 🎯 关键词：波动率、标准差、样本方差、年化平方根法则、波动率聚集、下行半方差、索提诺比率

## 这节课为什么重要

同样是「赚钱机器」，国债年化波动 2%，比特币 60%，差 30 倍——根本不是同一个游戏。这节课用「班级身高」这个生活类比建立波动率的数字直觉，讲懂六个核心工具：

1. **标准差 / 方差**：风险的数学定义；
2. **样本方差与贝塞尔修正**：为什么除以 N−1；
3. **年化波动率**：σ_ann = σ_daily × √252 的平方根法则；
4. **波动率聚集**：大震之后有余震；
5. **下行半方差与索提诺比率**：把「坏波动」和「好波动」分开。

学完你拿到任何一只股票 / 基金 / 币，先看一眼波动率，就知道这个游戏自己扛不扛得住，不会再在大震荡里割肉割在最低点。

## 学习目标

- [x] 理解金融对「风险」的精确定义：不是亏多少，而是收益的离散程度（用标准差度量）
- [x] 掌握样本方差 vs 总体方差的区别，以及贝塞尔修正为什么除以 N−1 而不是 N
- [x] 记住年化波动率公式 σ_ann = σ_daily × √252，理解为什么是 √N 而不是 N
- [x] 看清「波动率聚集」（volatility clustering）的金融常态：大震后有余震，平静期可持续
- [x] 学会用下行半方差 + 索提诺比率，把「坏波动」和「好波动」分开，贴近真实交易心理
- [x] 用波动率反向定仓位：高波动小仓，低波动可重仓，先有「游戏难度」再谈下注大小

## 一点历史八卦

- **1952 年**，25 岁的 Harry Markowitz 在博士论文里第一次把股票收益的「波动率」当成风险的数学度量，提出现代组合理论（Mean-Variance Optimization），1990 年因此拿了诺贝尔经济学奖。这个「收益看均值、风险看标准差」的二分法，统治了之后 70 年的资产管理行业。
- **1987 年 10 月 19 日**「黑色星期一」，标普 500 一天跌 22.6%。按此前几年约 1% 的日波动率推算，这是一个 22 倍标准差事件，概率约 10 的负 109 次方——比宇宙原子总数的倒数还小。正态分布说这种事一辈子不该发生，但市场后来又有 1998 LTCM、2008 雷曼、2020 新冠。标准差告诉你「平时多大震」，「极端有多极端」要另算。
- **2015 年 6 月** A 股股灾，沪深 300 三个月跌 43%，无数散户在 7 月初千股跌停那一周割肉割在最低点——他们不是不知道股票会跌，是没有事先在脑子里跑过一遍「年化波动 16–18% 意味着一年里真可能跌 30%+，我能不能扛」。
- **2003 年**，Robert Engle 因 GARCH 模型（量化波动率聚集）拿诺贝尔奖——「大震后有余震」从经验变成了科学。

> [!TIP]
> 本节最重要的两句话：**风险等于散度，不等于亏损；事先看一眼年化波动率，你就知道自己能不能扛。**
