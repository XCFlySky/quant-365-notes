# ② 因子投资派与三派对比

> 散户的主战场在这里：工具免费、容量灵活、1 万块就能开始。

## 因子投资派：长期 alpha 的工业化

**核心思想**：某些「特征」（因子）在历史上长期解释股票收益。系统性地买「好因子」的股票、卖「差因子」的，长期就有 alpha。

经典五因子：**价值、动量、质量、低波、规模**。

| 维度 | 内容 |
| --- | --- |
| 数学工具 | 线性回归、面板数据、IC/IR 分析、Barra 模型、ML（XGBoost 等） |
| 资金容量 | **最大**（几百亿到几千亿），换手率低，适合大资金 |
| 代表机构 | AQR、Dimensional(DFA)、BlackRock、Robeco |
| **散户可达性** | **最高**——对延迟不敏感，Tushare/akshare/sklearn 全免费 |

> [!IMPORTANT]
> 因子投资是这套 365 天课程的**主推方向**（课程权重 60%），因为它是对散户唯一「重武器免费」的流派。

## 三派对比总表

| 维度 | 高频做市 | 统计套利 | 因子投资 |
| --- | --- | --- | --- |
| 资金容量 | 1–50 亿 | 5–300 亿 | 50–3000 亿 |
| 决策速度 | 微秒 | 秒到日 | 周到月 |
| 团队规模 | 30–100 人 | 50–300 人 | 200–1000+ 人 |
| 技术栈 | C++/FPGA | Python+C++ | Python+SQL |
| 散户友好度 | ⛔ 0 | 🟡 中 | ✅ 高 |
| 本课程权重 | 10% | 30% | 60% |

> [!NOTE]
> 一个反直觉的规律：**技术栈越高级、工程门槛越高，资金容量反而越小**——这是市场结构决定的，不是谁不够聪明。

## 散户的选择与融合趋势

**散户的优先级**：

1. **因子投资派**（最高优先）：工具免费，1 万起步；
2. **统计套利派**（次优先）：配对/ETF 套利可做，加密 funding 套利尤其友好；
3. **高频做市派**：不进。但要学它的微结构知识——不为做高频，为**不被高频割**。

**融合趋势**：Renaissance 同时用统计套利 + 因子 + 部分高频；AQR 用 ML 改造传统因子；中国的指数增强 = 因子选股 + 统计套利对冲 + 高频成交优化。**最终你也会跨派——但起步先专一一派，深扎下去再融合。**
