# ⑤ 避坑指南与实战规则

> 选流派就像选专业：选错了不是不努力，是战场根本不属于你。

## 五个常见的坑

### ⚠️ 坑 1：想做高频但没基础设施

**现象**：看完 Virtu 的故事就想做高频。
**真相**：没有 co-location + 专线 + FPGA + C++ 团队，你比机构慢 10000 倍，挂的单全是机构的对手盘。99% 的散户「高频策略」其实是中频陷阱。

### ⚠️ 坑 2：做统计套利但股票池太小

**现象**：只在自己熟悉的 50 只股票里找配对。
**真相**：统计上不显著。配对交易需要 200–500 只候选两两组合 → 跑协整 → 留下 5–10 对。池子太小，信号太少。

### ⚠️ 坑 3：因子选股不做中性化

**现象**：纯按 PE 选股，结果选出来的全是小盘股。
**真相**：你以为赚的是价值因子的钱，其实赚的是规模因子的钱——**归因混乱**。任何因子研究必须做行业 + 市值中性化（P3 阶段会细讲）。

### ⚠️ 坑 4：跟风换方向

**现象**：听说 ML 赚钱就学深度学习，听说加密火就转加密。
**真相**：每派都要 1–2 年才小成，频繁切换让你永远在「入门状态」。

### ⚠️ 坑 5：看不见融合趋势

**现象**：纯因子派觉得统计套利是旁门，纯套利派觉得因子太慢。
**真相**：头部机构早就跨派打了。365 天的目标是**三派都懂、主攻一派**。

## 散户选派 SOP

| # | 规则 |
| --- | --- |
| 01 | 起步 0–3 个月：**专攻因子投资派**（工具免费、可达性最高） |
| 02 | 3–6 个月：加学统计套利的简单形态（配对 + ETF 套利） |
| 03 | 6–12 个月：学高频派的微结构知识——不为做，为不被割 |
| 04 | 选派看时间：每天 1 小时 → 因子；每天 2–3 小时 → 可加统计套利；高频不是时间问题 |
| 05 | 永远不在自己睡觉的时区做高频；永远不用「听懂的」高频策略实盘 |
| 06 | 兴趣在数学/物理/工程 → 统计套利更配；在分析/直觉/产业 → 因子+主观更配 |
| 07 | 第一年专注因子，第二年加统计套利，高频留作认知 |

> [!IMPORTANT]
> 选派最重要的不是「哪派最赚」，而是「**哪派和你的时间、资源、兴趣最匹配**」。
