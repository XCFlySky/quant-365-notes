# ① 统计套利派与高频做市派

> 一个赚「橡皮筋弹回来」的钱，一个赚「千万人过路费」的钱。

## 统计套利派：寻找均值回归

**核心思想**：某些资产的价格关系像一根橡皮筋——短期会被拉开，但总会弹回来。于是在偏离时做多低估的、做空高估的，等回归就赚了。

| 维度 | 内容 |
| --- | --- |
| 代表策略 | 配对交易、ETF 套利、跨期套利、统计因子 |
| 数学工具 | 协整检验、卡尔曼滤波、PCA、因子分解 |
| 资金容量 | 中等（几亿到几百亿），A 股头部量化私募主战场 |
| 速度 | 中频，日级到分钟级 |
| 技术门槛 | 中等，Python + 一点 C++ 够用 |
| **散户可达性** | **可以做**（配对/ETF 套利），但成本要算清楚 |

> [!NOTE]
> 中国的「量化中性」「指数增强」产品，基本都是这一派的工业化版本（九坤、平安系指增等）。

## 高频做市派：赚价差和流量

**核心思想**：在交易所同时挂买单和卖单，赚每一笔的买卖价差——像开收费站。同时承担库存风险，靠快速对冲管理。

| 维度 | 内容 |
| --- | --- |
| 代表机构 | Citadel Securities、Jane Street、Virtu、Tower、Optiver |
| 数学工具 | 订单簿建模、库存管理（Avellaneda-Stoikov）、贝叶斯更新 |
| 资金容量 | 小到中（受流动性限制），但夏普可以非常高（旗舰策略 5+） |
| 技术门槛 | **极高**：FPGA / co-location / 专线 / C++，资本支出动辄上千万 |
| **散户可达性** | **几乎为零** |

> [!WARNING]
> Virtu 2014 年披露：连续 1238 个交易日里只有 1 天亏钱。这种胜率不是预测能力，是**结构性吃流量**的胜利——而散户恰恰是他们流量的一部分。这一派要「懂」，但不要「做」。
