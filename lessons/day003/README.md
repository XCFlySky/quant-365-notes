## 这节课为什么重要

「量化」这个词下面其实藏着三个世界：有人靠**均值回归**赚钱，有人靠**速度和流量**赚钱，有人靠**长期因子**赚钱。它们的数学工具、资金容量、团队结构、技术栈差别巨大——当然，**对散户的友好程度也天差地别**。这节课帮你把三派看清楚，然后选定主攻方向。

## 学习目标

- [x] 看懂三派对资金量、技术延迟、团队规模的不同要求
- [x] 找出哪一派对散户友好、哪一派散户进去就是送钱
- [x] 选定接下来 3–6 个月的主攻方向

## 三段历史，三个「量化」

- **1980 年代**，Morgan Stanley 的 Tartaglia 团队凌晨用大型机跑配对交易——**统计套利派**的源头，后来走出 D.E. Shaw 和 Renaissance；
- **1990 年代末**，Citadel、Jane Street 以电子化做市 + 速度优势赚价差——**高频做市派**崛起，如今美股 50% 以上的成交量由高频做市商提供；
- **1990 年代**，Fama & French 提出三因子模型，AQR（1998）把学术理论工业化——**因子投资派**从此成为机构标配。
