# Day 001 · 课程总览：365 天能学到什么

> 这不是一节赚钱课，这是一节「决定要不要赚这个钱」的课——先把整个量化金融的版图看清，再决定要不要上路。

- 📂 阶段：P1 · 量化基础
- ⏱️ 时长：约 18 分钟
- 🎯 关键词：量化行业全景、11 阶段学习路径、工具与社区资源

## 这节课为什么重要

如果把 365 天的学习比作一次长途自驾，这节课就是**出发前摊开地图的那 25 分钟**。看完你会搞明白三件事：

1. **量化到底是什么**（不是玄学，也不是印钞机）；
2. **散户做量化能不能赚钱**（能，但有明确的边界）；
3. **自己要不要走这条路**（不适合也是一份有价值的答案）。

## 学习目标

- [x] 搞清楚量化金融的边界：它能解决什么、解决不了什么
- [x] 看懂这 365 天的整体地图，知道每个阶段你会进化成什么样
- [x] 评估自己的起点（代码、数学、资金、时间），定一个现实的节奏
- [x] 建立量化思维的第一块砖：**用概率和规则代替直觉**
- [x] 了解一名散户做量化的真实成本（钱、时间、心力）

## 一点历史八卦

- **1971 年**，Ed Thorp 用《Beat the Dealer》把概率论从赌场带进华尔街，被称为「量化交易之父」；
- **1982 年**，数学家 James Simons 创立文艺复兴科技（Renaissance），旗舰基金 Medallion 做到几十年年化 35%（扣费后）的传奇；
- **1990 年代**，LTCM 让全世界看到量化的极致与脆弱；
- **2018 年之后**，散户窗口真正打开：Tushare/akshare 免费数据、Python 生态成熟、云服务器几十块一个月。今天一个本科生在自己电脑上能做的事，十年前一个机构需要一支团队。

> [!TIP]
> 这个窗口不会永远开着，但现在它是开的。
