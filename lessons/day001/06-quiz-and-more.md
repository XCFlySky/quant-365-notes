## 自测题

### Q1：你能用 30 秒向一个完全不懂的朋友讲清楚「什么是量化交易」吗？

<details>
<summary>💡 答题思路提示（先自己想，再点开）</summary>

抓三个词：**数据、规则、执行**。示例答案：「量化交易就是把买卖股票的决定写成明确的规则（比如『买过去一个月涨得最多的 20 只』），用干净的历史数据验证这个规则过去是否赚钱，然后让电脑机械地执行它，不凭感觉、不临盘改主意。」回「[① 量化是什么](lessons/day001/01-what-is-quant.md)」复习。

</details>

## 一页总结（建议截图，3 天后再看一遍）

1. 量化 = 数据 + 模型 + 工程，缺一不可，纪律是指数；
2. 散户量化的窗口是真实的：中低频策略 + 小资金 + 严格风控 = 长期可活，但别碰高频；
3. 365 天分 11 个阶段：基础 → 工具 → 因子 → 策略 → 回测 → ML → 衍生品 → 风控 → HFT → 实盘 → 前沿；
4. 起步只需免费工具，3 个月后再考虑付费；
5. 1.01^365 ≈ 38，差距全在「每天」；
6. 五大坑：急于实盘 / 追神 / 不写代码 / 不记笔记 / 孤军奋战。
