# Day 011 · 厚尾与黑天鹅

> 金融市场永远会出现你模型里说不可能的事——给它留余地。正期望赌局的最优仓位反直觉地小，败在过大仓位的人比败在选错方向的人多得多。

- 📂 阶段：P1 · 量化基础
- ⏱️ 时长：约 22 分钟
- 🎯 关键词：学生 t 分布、幂律分布、凯利公式、杠铃策略、反脆弱

## 这节课为什么重要

上一课我们看到正态分布在金融的尾巴上完全失效——沪深三百五年里发生两次按正态算一万年才一次的暴跌。这一课追问下一个问题：**既然正态不行，拿什么替代？** 三件事：

1. **学生 t 分布**——只比正态多一个自由度参数，但拟合金融数据的精度高出一个数量级（标普实测自由度仅 ≈ 4）；
2. **凯利公式**——数学告诉你正期望赌局的最优仓位，反直觉地小（胜率 55% 的赌局最优只下 10%）；
3. **塔勒布的反脆弱思想 + 杠铃策略**——九成极保守 + 一成极冒险，黑天鹅来时不死还能赚。

课程用真实标普五百数据做学生 t 拟合，跑 5000 次一年期杠铃 vs 单一仿真、1000 次 × 200 路径凯利仿真。学完你以后看到极端行情就不会束手无策。

## 学习目标

- [x] 理解学生 t 分布的自由度参数（ν），知道 ν→∞ 退化为正态、ν=4 对应金融典型厚尾
- [x] 看懂学生 t 分布在 ±3/4/5/6σ 上对真实数据的拟合精度比正态高几千倍
- [x] 认识幂律分布，理解金融市场尾巴可能比任何标准分布都厚
- [x] 掌握凯利公式 f* = (pb−q)/b，以及为什么实战只用半凯利或四分之一凯利
- [x] 理解杠铃策略的反脆弱结构 = 90% 极保守 + 10% 极冒险，黑天鹅日反而暴利
- [x] 记住散户三大法则：个人版杠铃 / 半凯利仓位 / 永远留 20% 黑天鹅基金

## 一点历史八卦

- **1908 年**，吉尼斯啤酒厂的统计学家 Gosset 发现小样本下正态拟合不准，推导出多一个「自由度」的新分布；公司不让用真名发论文，他用笔名「Student」——这就是学生 t 分布名字的由来。一个为啤酒厂服务的工具，百年后成了金融风控的新地基；
- **1956 年**，贝尔实验室的 John Kelly 推出凯利公式，本意是解决信道编码问题，后来被赌徒和华尔街广泛使用；Ed Thorp 把它用到 21 点和股市，Buffett 和 Munger 也公开赞过；
- **1963 年**，Mandelbrot 首次提出股价波动遵循幂律；
- **2008 / 2020 年**，塔勒布与 Spitznagel 的 Universa 基金靠长期持有价外看跌期权，在两次黑天鹅中暴赚（据报道 2008 年获利数十亿美金，2020 年 3 月单月 36 倍）。
