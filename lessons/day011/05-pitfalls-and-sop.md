# ⑤ 避坑指南与实战 SOP

> 黑天鹅最爱的人：重仓抄底的、加杠杆的、伪分散的、嫌期权贵的、拿回测胜率直接套凯利的。

## 坑 01：觉得这一波跌已经够深、不会再跌

- **现象**：大盘跌了 15%，觉得「到底了」，重仓抄底。
- **真相**：厚尾分布意味着**大跌之后大概率还会再跌**——引发大跌的根本原因往往还在发酵。2008 雷曼倒下后大跌持续 2 个月才见底；2020 新冠熔断后又跌了 3 周。
- **避坑动作**：永远不要在第一波大跌时就重仓抄底。黑天鹅日不要急，等波动率下来再分批。

## 坑 02：觉得「我这一次可以承受」，所以加杠杆

- **现象**：自信扛得住波动，上 2–3 倍杠杆放大收益。
- **真相**：黑天鹅最爱给加杠杆的人上课。LTCM 1998 加了约 30 倍杠杆，诺奖团队两周内输光 90%。无杠杆时极端日跌 7%，3 倍杠杆就是跌 21%，直接爆仓。
- **避坑动作**：记住这句话——**杠杆不是放大收益，是放大破产概率。** 无杠杆是底线。

## 坑 03：以为分散持有十几只股票就分散了风险

- **现象**：持有 30 只 A 股，觉得鸡蛋不在一个篮子里。
- **真相**：黑天鹅来时，大部分股票同涨同跌（Day 8 讲过，极端时相关性飙到 0.9+）。你以为的 30 只分散组合，极端日就是 1 只。
- **避坑动作**：真正的分散是**跨资产类别**（股 + 债 + 黄金 + 加密），而不是同一类资产的多只。

## 坑 04：认为期权太贵，不划算

- **现象**：看到价外期权每月损耗 0.5%，觉得「白送钱」。
- **真相**：期权的「贵」恰恰是它价值所在——你买的是黑天鹅日的保险，平时损耗就是保费。Universa 基金每年正常情况亏 5–10%，但 2008 / 2020 两次黑天鹅赚 100+ 倍，长期年化正收益。
- **避坑动作**：把期权支出当**保费**预算，而不是当亏损。小仓位、长期、纪律性持有。

## 坑 05：凯利公式按回测胜率使用，不打折

- **现象**：回测胜率 60%，高高兴兴按公式用 f* = 33% 的仓位。
- **真相**：实战胜率永远低于回测——滑点、手续费、数据偏差、过拟合全在吃你的真实胜率。按回测值满额凯利，等于把模型误差也加了杠杆。
- **避坑动作**：专业交易员对凯利的态度是——**当成上限，实际仓位永远在它之下**。标准做法：只用半凯利甚至四分之一凯利。

## 应对厚尾的实战 SOP

| # | 规则 |
| --- | --- |
| 01 | 尾部风险估计永远用学生 t（ν=4）或历史模拟，不用正态 |
| 02 | 永远假设单日跌幅可能是正态预测的 3–5 倍 |
| 03 | 凯利公式只当上限用，实战只下半凯利或四分之一凯利 |
| 04 | 个人版杠铃：60% 安全 + 30% 中等 + 10% 高赔率 |
| 05 | 永远留 20% 黑天鹅基金（货币基金 / 短债），只在大跌 −20%+ 时动用 |
| 06 | 黑天鹅日不要急着抄底——厚尾意味着大跌后大概率还有大跌 |
| 07 | 无杠杆是底线，加杠杆需要极强的非对称对冲（如长期价外认沽） |
