# ⑤ 避坑指南与实战规则

> 这一课的坑，核心都指向一件事：认清自己的天性，别两头不靠。

## 五个常见的坑

### ⚠️ 坑 1：把量化神话化

**现象**：以为量化 = 稳赚。
**真相**：量化一样大幅回撤，只是回撤来源不同。Renaissance 也亏过钱，LTCM 直接 21 天爆掉。任何承诺稳赚的都是骗子。

### ⚠️ 坑 2：把主观浪漫化

**现象**：看完巴菲特传记觉得自己也能复制。
**真相**：99% 的主观投资者跑不赢指数。主观的难度在「坚持」和「认知」——对绝大多数人都是不可能任务。

### ⚠️ 坑 3：二选一陷阱

**现象**：认为必须纯量化或纯主观。
**真相**：多数顶级投资人是**混合派**——用直觉发现机会，用规则控制规模和风险。最聪明的散户也该这么做。

### ⚠️ 坑 4：用错地方

**现象**：拿主观风格做高频（注定亏），拿量化方式做巨额并购套利（样本太小）。
**避坑动作**：两种风格各有适用区，选错了就是把锤子当螺丝刀。

### ⚠️ 坑 5：散户两头不靠（最常见）

**现象**：既没有主观投资人 10 年的深度认知，也没有量化的工程纪律。结果用情绪做决策（主观失败）、用浅薄规则做交易（量化失败）。
**避坑动作**：要么补深度，要么补纪律——**最忌两边都是半吊子**。

## 量化 + 主观混合 SOP

| # | 规则 |
| --- | --- |
| 01 | **用主观提假设**：任何想法先想清「为什么会赚钱」，形成经济直觉 |
| 02 | **用量化验证假设**：把直觉写成规则，跑历史回测 |
| 03 | **用量化做规模**：验证通过后机械执行，不让情绪干扰仓位 |
| 04 | **用主观做边界**：极端行情/政策冲击超出样本时，人工介入 |
| 05 | **短期相信规则**：1–3 个月内不质疑模型，即使在亏损 |
| 06 | **长期质疑规则**：1–3 年定期 review，过期的规则及时下架 |
| 07 | **只做能解释的策略**：夏普 5.0 但说不清原理的，大概率是过拟合，远离 |

> [!IMPORTANT]
> 一句话心法：**用主观提假设、做边界；用量化做验证、做规模、做风控。**
