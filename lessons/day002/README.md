# Day 002 · 量化交易 vs 主观交易

> 巴菲特和 Simons 都是史上最伟大的投资者，但方法论几乎对立。这节课不站队，只帮你看清：你的天性更适合哪一种。

- 📂 阶段：P1 · 量化基础
- ⏱️ 时长：约 18 分钟
- 🎯 关键词：决策依据、容量与速度、可复制性

## 这节课为什么重要

这是一节**对照课**。量化交易和主观交易是两条本质不同的路——决策怎么做、钱能赚多大、错了往哪儿错、能不能传给别人，全都不一样。看完你应该能判断：自己更适合做巴菲特，还是做 Simons，还是**先用量化的纪律规范自己的主观判断**（大多数人的答案）。

## 学习目标

- [x] 用四个维度讲清量化与主观的本质差异：决策依据、容量、速度、可复制性
- [x] 看懂为什么资金一旦做大，管理人几乎都被迫走向量化或半量化
- [x] 识别散户最常见的「两头不靠」陷阱
- [x] 评估自己的性格，选一个起步路径（纯主观 / 主观+量化 / 纯量化）
- [x] 建立心法：**用主观提假设，用量化验证；用直觉发现机会，用规则控制规模**

## 两种伟大的赚钱方式

- **巴菲特**：1965–2024 年化约 19.8%，核心是少数大重仓 + 极长持有 + 深度认知。他的 alpha 是「懂这家公司，懂得比绝大多数人深」；
- **Simons**：Medallion 基金 1988–2018 年化约 39%（扣费后 35%），只对内部员工开放。它的 alpha 是「每天找到 5000 个微小信号，每个胜率 51%，但每天交易 100 次」。

> [!NOTE]
> 问题不是哪种方法「对」，而是哪种**适合你**——以及你能不能认清天性之后，坚持 10 年。
