# ② 偏度、峰度与 Q-Q 图

> 标准差只说「散得多开」，没说「往哪边散、尾巴有多厚」。偏度和峰度补上这两个维度，Q-Q 图让你十秒看穿一切。

## 偏度（Skewness）：分布歪向哪边

偏度衡量**对称性**——钟形曲线是不是左右对称：

- **偏度 = 0**：完美对称（正态就是这样）
- **负偏度**：左尾巴长 → 大跌的概率比大涨高，**大多数股票指数是这种**
- **正偏度**：右尾巴长 → 大涨的概率比大跌高，大宗商品偶尔出现

白话记忆法：偏度的符号 = 长尾巴的方向。负偏度就是「往下栽的风险更大」。

## 峰度（Kurtosis）：尾巴有多厚

课程用的是**超额峰度**（正态 = 0），衡量**尾部厚度**：

- **超额峰度 = 0**：跟正态一样厚
- **正超额峰度**：厚尾——中间更尖、尾巴更厚，极端值比正态预测多
- **负超额峰度**：薄尾

四标的实测（课程样本，近 5 年日收益）：

| 标的 | 偏度 | 超额峰度 | 白话解读 |
| --- | --- | --- | --- |
| 沪深三百 | ≈ 0.21 | ≈ 7.09 | 厚尾明显 |
| 标普五百 | ≈ 0.16 | ≈ 7.17 | 同样厚尾 |
| 纳斯达克 | ≈ 0.21 | ≈ 5.54 | 厚尾 |
| 比特币 | ≈ 0.30 | ≈ 3.44 | 厚尾（课程口径：四个标的全部厚尾） |

> [!WARNING]
> **两个收益序列可以标准差完全相同，但峰度差 10 倍，风险天差地别**——一个偶尔暴跌让账户清零，另一个大致稳定。光看 Sharpe 不看峰度 = 裸奔。

## Q-Q 图：十秒识别厚尾的瑞士军刀

Q-Q 图（分位数对比图）是量化界判断分布形态的「瑞士军刀」。

**怎么画：** 横轴 = 正态分布的理论分位数，纵轴 = 实测数据的分位数，每个数据点对应一个 (理论 q, 实测 q)。

```text
理论 q_i = Φ⁻¹((i − 0.5) / N)
实测 q_i = sorted_data[i]
```

**怎么读：**

- 数据是正态 → 所有点规规矩矩落在 **45° 直线**上；
- 有厚尾 → 两端的点偏离直线：**左下角点比直线低、右上角点比直线高**，形状像一条 S 曲线或两端翘起的「弓」。

金融实测（课程样本）：

| 标的 | Q-Q 图表现 |
| --- | --- |
| 沪深三百 | 两端明显翘起，典型厚尾 |
| 标普五百 | 两端翘起，但比沪深略缓 |
| 纳斯达克 | 右上角异常高（科技股大涨多） |
| 比特币 | 两端直接甩出图外——根本不是同一个游戏 |

一行代码就能画：

```python
from scipy import stats
stats.probplot(returns, dist='norm', plot=plt)   # 一行画出 Q-Q 图
```

> [!IMPORTANT]
> 实战铁律：**你拿到任何一个新策略的收益序列，第一件事就画 Q-Q 图。** 十秒钟就能判断能不能套用经典正态模型。看到两端不在直线上立刻警觉——你的模型假设可能不成立，需要换学生 t 分布或更复杂的工具（Day 011 的主题）。

## 四个工具怎么配合

| 工具 | 回答的问题 | 速度 |
| --- | --- | --- |
| 标准差 | 日常波动多大 | 一眼 |
| 偏度 | 大跌风险 vs 大涨机会哪个大 | 一眼 |
| 峰度 | 极端日比正态多多少 | 一眼 |
| Q-Q 图 | 整体形状哪里偏离正态 | 十秒 |

> [!TIP]
> 一句话总结：**标准差告诉你天气，偏度和峰度告诉你气候，Q-Q 图直接给你看卫星云图。**
