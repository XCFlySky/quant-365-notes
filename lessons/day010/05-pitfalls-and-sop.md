# ⑤ 避坑指南与实战 SOP

> 正态分布在金融里的坑，全都围绕一句话：它在尾巴上撒谎，而你的本金恰恰死在尾巴上。

## 坑 01：把过去最大回撤当未来上限

- **现象**：看到自己策略过去 3 年最大回撤 −10%，就以为未来撑死也是这个数。
- **真相**：厚尾意味着未来**一定有更深的**——历史从来不是上限，只是样本。
- **避坑动作**：用学生 t 分布 / 极值理论（EVT）估计 99% 分位数风险，而不是拿历史最大值当天花板。

## 坑 02：以为高频数据更接近正态

- **现象**：觉得把数据切细（分钟级、秒级）就能更接近正态假设。
- **真相**：**正好相反**——日内高频数据厚尾更严重，因为短时间的流动性踩踏比日级数据剧烈得多。
- **避坑动作**：明白为什么 Renaissance、Two Sigma 这类做高频策略的对冲基金，风控比低频严格得多。频率越高，越不能信正态。

## 坑 03：把样本均值当总体均值

- **现象**：3 年数据算出沪深三百日均收益 0.001%，就当成「市场的真实期望」。
- **真相**：这个估计本身有巨大误差，下一个 3 年可能完全不一样。
- **避坑动作**：做夏普比率 / 阿尔法估计时一定要看**置信区间和 t-stat**，不要把点估计当成真理。

## 坑 04：只看标准差不看偏度峰度

- **现象**：两只基金标准差都是 8%，就觉得风险一样。
- **真相**：两个收益序列可以标准差完全相同，但峰度差 10 倍，风险天差地别——一个偶尔暴跌让账户清零，另一个大致稳定。
- **避坑动作**：看任何收益序列，标准差、偏度、峰度三个数一起看。**光看 Sharpe 不看峰度 = 裸奔。**

## 坑 05：VaR 按正态算，严重低估真实风险

- **现象**：在险价值（VaR）是华尔街最爱的风险指标，按正态算沪深三百 99% VaR ≈ −2.5%/天，听起来很安全。
- **真相**：实际过去 5 年最大单日跌幅 −7%。如果你按正态 VaR 配仓再加杠杆，大跌那天直接爆仓。
- **避坑动作**：实战用**历史模拟法 / Cornish-Fisher 修正 / 极值理论**，正态 VaR 最多当个下限参考。

## 正态分布在金融的使用 SOP

| # | 规则 |
| --- | --- |
| 01 | 正态只在中段（±2σ 内）大致成立，尾部（±3σ 外）完全失效 |
| 02 | 做风险估计永远不要只用正态，用学生 t / EVT / 历史模拟做交叉验证 |
| 03 | 拿到新数据第一件事画 Q-Q 图，10 秒识别厚尾 |
| 04 | VaR 用正态算会严重低估真实风险，实战要历史模拟 + Cornish-Fisher 修正 |
| 05 | 看一只股票要算偏度 + 峰度，不能只看标准差 |
| 06 | 中心极限定理在长周期（>60 天聚合）可用，但散户交易是日级，日级 ≠ 正态 |
| 07 | 永远假设单日跌幅可能是正态预测的 3–5 倍，所以仓位永远不要满到极限 |

## 散户三条铁律

1. **永远不要满仓**——留 20% 浮动空间；
2. **永远准备 20% 黑天鹅基金**——货币基金 / 短债，只在大跌时动用；
3. **大跌别摊平**——厚尾意味着大跌之后大概率还有大跌。
