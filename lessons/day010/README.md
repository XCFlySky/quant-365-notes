# Day 010 · 正态分布与金融

> 正态分布最大的危险不是它错，是它太好用——好用到让所有人都忘了它的边界。它解释了 99% 的中段，但能让你账户归零的就是它解释不了的那 1%。

- 📂 阶段：P1 · 量化基础
- ⏱️ 时长：约 20 分钟
- 🎯 关键词：68-95-99.7 法则、偏度与峰度、Q-Q 图、正态假设的代价、LTCM

## 这节课为什么重要

正态分布是金融教科书最爱的工具，也是整个现代金融工程的地基。但看看这些数字：

- **1987 年 10 月 19 日**，标普五百一天跌 22.6%（黑色星期一），按正态分布算 10⁻¹⁰⁹ 年才一次，而它在两百年里已发生四五次；
- **沪深三百 2024-10-09 和 2025-04-07** 两次跌 7%（均为 −6.37σ 事件），按正态算一万年一次，我们五年里发生了两次。

这节课用沪深三百、标普五百、纳斯达克、比特币四个标的的真实数据，讲透正态在金融**中段为什么有效、尾部为什么完全失效**；讲清 68-95-99.7 法则、偏度、峰度、Q-Q 图四个工具；复盘 LTCM 1998 和雷曼 2008 两个被正态滥用搞死的真实案例。学完你会对任何「假设收益服从正态」的金融模型，本能在心里画一个问号。

## 学习目标

- [x] 理解正态分布（钟形曲线）的两个参数：均值 + 标准差，以及它在自然界为何无处不在
- [x] 记牢 68-95-99.7 法则，以及它在金融中段实测大致成立、在尾部大幅失效
- [x] 用偏度（对称性）+ 峰度（厚尾度）量化分布形状，知道正态偏度=0 / 超额峰度=0
- [x] 学会画 Q-Q 图（分位数对比图）一眼识别厚尾，十秒判断能不能套用经典模型
- [x] 理解 LTCM 1998 和雷曼 2008 是怎么被正态分布的滥用搞死的
- [x] 掌握散户应对厚尾的三条铁律：不满仓 / 留备用金 / 大跌别摊平

## 一点历史八卦

- **1809 年**，Gauss 在《天体运动理论》里第一次系统化正态分布，用来描述天文测量误差，所以正态分布也叫高斯分布；
- **19 世纪**，Quetelet 把它搬到社会统计（身高、智商）；**1900 年**，Bachelier 首次用正态建模股价；**1973 年**，Black-Scholes-Merton 期权公式也建立在正态假设上；
- **1963 年**，Mandelbrot 首次质疑金融正态假设，提出幂律分布——这个「异端」要等到 Day 011 才真正翻案；
- **1998 年**，LTCM（合伙人里有 Scholes 和 Merton 两位诺奖得主）因策略全押在正态假设上，几个月内净值蒸发 90%，需要美联储召集 14 家投行救市。
