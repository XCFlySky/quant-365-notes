# ① 协方差与协方差矩阵

> Day 007 的标准差量「一只资产自己怎么跳」，协方差量「两只资产是不是一起跳」。

## 协方差：一起动，还是对着动？

**方差**是把每只资产的「偏离量」自己乘自己；**协方差**（covariance）是把两只资产的偏离量**互相乘**再求平均：

```text
Cov(X, Y) = Σ (X_i − X̄)(Y_i − Ȳ) / (N−1)
```

白话解读：

- 两只资产**同涨同跌**的日子里，两个偏离量同号，乘积为正 → 协方差 > 0；
- **一个涨一个跌**的日子多，乘积为负 → 协方差 < 0；
- **各走各的**，正负抵消 → 协方差 ≈ 0。

所以协方差的**符号**告诉你方向（一起动还是对着动），**大小**告诉你强度。方差其实就是协方差的特例：`Cov(X, X) = Var(X)`。

## 协方差的毛病：单位太怪

协方差有个实用问题：它的数值大小取决于两只资产自身的波动幅度。「Cov = 0.0004」是大是小？没人能直接看懂。

解决办法是**标准化**——把协方差除以两只资产各自的标准差，得到**相关系数**（correlation）：

```text
ρ(X, Y) = Cov(X, Y) / (σ_X · σ_Y)     ∈ [−1, +1]
```

- **ρ = +1**：完全同向（复制粘贴）；
- **ρ = 0**：线性上无关；
- **ρ = −1**：完全反向（完美对冲）。

> [!NOTE]
> 相关性就是把协方差压缩到 −1 到 +1 的标准化版本。以后说「两只股票相关性 0.57」，指的就是这个 ρ。

## 协方差矩阵：N 只资产的「关系总表」

3 只以上资产时，把两两协方差排成一个方阵，就是**协方差矩阵** Σ：

```text
        [ Var(A)   Cov(A,B)  Cov(A,C) ]
   Σ =  [ Cov(B,A)  Var(B)   Cov(B,C) ]
        [ Cov(C,A)  Cov(C,B)  Var(C)  ]
```

- 对角线上是各资产自己的方差；
- 非对角线是两两协方差，矩阵关于对角线对称（Cov(A,B) = Cov(B,A)）。

这张表是**一切组合风险计算的数学基础**——Markowitz 的组合优化、风险平价、因子模型，全都从它出发。

## 组合波动的公式：wᵀΣw

有了协方差矩阵，N 只资产按权重 w 组成的组合，其方差是：

```text
σ_p² = wᵀ Σ w
σ_p  = √(wᵀ Σ w)
```

这个公式藏着分散化的全部秘密：组合方差**不是**各资产方差的加权和，而是多了两两协方差那一堆交叉项。

- 资产间相关性高（交叉项大且正）→ 组合方差 ≈ 各资产方差平均 → **没分散**；
- 资产间相关性低或为负（交叉项小或负）→ 组合方差被显著拉低 → **真分散**。

> [!IMPORTANT]
> 这就是为什么「买了三只股票」不等于「分散了」：wᵀΣw 里的 Σ 如果满是大正数，三只股票和一只没区别。分散发生在协方差矩阵里，不发生在持仓数量里。
