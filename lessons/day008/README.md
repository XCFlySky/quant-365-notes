# Day 008 · 协方差与相关性

> 你以为买了三只不同的股票就分散了？如果三只都是新能源车赛道，你只买了一个篮子。

- 📂 阶段：P1 · 量化基础
- ⏱️ 时长：约 18 分钟
- 🎯 关键词：协方差、协方差矩阵、皮尔逊相关、斯皮尔曼秩相关、伪分散

## 这节课为什么重要

Day 007 讲了「单只资产的风险怎么算」，这节课往前走一步：**两只资产「一起动」的关系怎么算**。这个问题决定了你组合的分散是不是真的：

1. **协方差**：两只资产同向还是反向动的原始度量；
2. **皮尔逊相关系数**：把协方差标准化到 −1 到 +1，可比；
3. **斯皮尔曼秩相关**：用排名代替数值，抗极端值；
4. **协方差矩阵**：组合风险计算的数学基础。

课程用真实数据算了两个组合：**比亚迪+宁德+特斯拉**（三条新能源船）vs **比亚迪+中石油+黄金**（三个不相干的驱动逻辑），直接看「伪分散」的代价。

## 学习目标

- [x] 理解协方差的含义：两只资产收益「一起动」的方向与强度
- [x] 掌握相关性是 −1 到 +1 的标准化协方差，会读相关矩阵
- [x] 分清皮尔逊（线性相关）和斯皮尔曼（秩相关），知道什么时候用哪个
- [x] 看懂协方差矩阵是组合风险的基础：σ_p² = wᵀΣw
- [x] 识破「伪分散」：数量上的分散 ≠ 驱动逻辑上的分散
- [x] 记住两个警告：线性相关 ≠ 因果；相关性会在极端行情骤升
