# Day 015 · 时间序列与平稳性

> 几乎所有量化工具（回归、ARIMA、协整）都只接受「乖的数据」——这节课教你用三行代码判断手里的数据乖不乖，以及不乖的话怎么把它变乖。

- 📂 阶段：P1 · 量化基础
- ⏱️ 时长：约 20 分钟
- 🎯 关键词：平稳性、ADF 检验、差分、协整、配对交易

## 这节课为什么重要

时间序列与平稳性是金融数据分析的两个根本起点。你可以把这节课想象成**学开车前的「辨方向」训练**——分不清方向，油门踩得再猛也只会撞墙。

这节课讲清四件事：

1. **平稳性是什么**——一个序列的均值、方差、协方差不随时间漂移；
2. **ADF 检验怎么用**——一行代码给你一个 p 值，p < 0.05 就是平稳；
3. **价格几乎都不平稳、收益率几乎都平稳**——这就是为什么所有量化分析第一步必须算对数收益；
4. **协整**——两个非平稳序列的某种线性组合可能是平稳的，这是配对交易的理论根基。

学完你就能判断：任何时间序列能不能直接做回归，任何两个标的能不能做配对。今天用沪深300的价格 vs 收益率做教科书级对照，再用中信证券 + 华泰证券两家券商龙头实测协整——结果很反直觉，往下看。

## 学习目标

- [x] 理解时间序列的定义：按时间顺序的数据，以及为什么金融数据必须按时间序列而不是独立观察来分析
- [x] 掌握平稳性的三个要求：均值不变、方差不变、协方差只跟时间差有关
- [x] 会用 ADF 检验判断任何序列是否平稳：一行 statsmodels 代码 + p 值解读
- [x] 知道为什么金融价格几乎都不平稳、日收益率几乎都平稳——必先做对数收益的根本原因
- [x] 理解协整的本质：两个非平稳序列的线性组合可能是平稳的，配对交易的理论根基
- [x] 会用 Engle-Granger 协整检验判断两个标的能不能配对，并用实测数据看反直觉案例

## 一点历史八卦

- **1980 年代**，摩根斯坦利的物理学家 Nunzio Tartaglia 组建量化团队，发现同行业「双胞胎」股票（可乐 vs 百事、麦当劳 vs 汉堡王、福特 vs 通用）的价差有「回到均值」的特性——这是配对交易的起源，团队 1985–1989 年从十几人扩到二三十人，赚了很多钱；
- **1980 年代末**，Renaissance 的 Simons 团队独立开发了类似策略，但用 ADF 和协整检验做严格统计验证，只交易统计上确实平稳的 spread——这种纪律让他们从 1988 年开始连续三十年盈利；
- **1987 年**，Engle 与 Granger 奠定协整理论，2003 年获诺贝尔经济学奖；**1979 年**，Dickey 与 Fuller 提出 ADF 检验；
- **1998 年**，LTCM 倒闭——他们的协整检验只用了四到五年平静期数据，俄罗斯违约让一切协整关系瞬间崩溃，几个月亏掉 90% 资本。

> [!IMPORTANT]
> 历史反复讲同一句话：**协整不是永恒的关系，会随宏观环境变化**。教科书「应该协整」的对子，实战里可能根本不协整——实测才是真理，定期复检是纪律。
