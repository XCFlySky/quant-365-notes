# ⑤ 避坑指南与实战规则

> 这些坑你不主动避，迟早会主动找上你。每个坑都附避坑动作，不只是描述现象。

## 坑 1：拿原始价格做相关分析（没差分 = 假）

- **现象**：看两只股票「走势相似」，直接对原始价格做线性回归，得到 R² = 0.95，宣布「这俩协整，可以配对」。
- **真相**：两个完全独立的随机游走价格序列，普通回归也有 75% 概率出现 R² 高 + p < 0.05 的「伪回归」（Granger 1974）。价格序列都非平稳，回归结果不可信。
- **避坑动作**：固定三步走——① 各自 ADF 确认都非平稳；② `statsmodels.tsa.stattools.coint` 做 Engle-Granger 协整；③ 对 spread 自身做 ADF。三步全过才算数。

## 坑 2：以为「协整 = 相关性高」

- **现象**：两只股票 Pearson 相关系数 0.9+，就认为它们协整。
- **真相**：相关性高只说明两个序列「变化方向」同步，不代表 spread 平稳。举例：corr = 0.95 但两只股票均值漂移速度不同 → spread 持续扩大 → 不协整。
- **避坑动作**：记住协整测的是 **spread 的平稳性**，不是相关性。Pearson 相关和协整是两个独立指标，相关高得不出协整结论。

## 坑 3：协整窗口太短（< 250 天不可信）

- **现象**：用过去 60 天或 100 天数据测协整，p 值显著，兴冲冲上线。
- **真相**：样本量太小，ADF 检验没有统计效力，「显著」只是噪声。LTCM 倒闭的部分原因就是只用了四到五年平静期数据——样本里没有极端事件，协整看起来稳健，实际脆弱。
- **避坑动作**：窗口最少 1 年（252 个交易日），稳健用 3–5 年（750–1250 天），且**必须包含极端时段**。

## 坑 4：不复检——协整关系会随时间变化

- **现象**：配对上线后放着不管，半年后莫名持续亏损。
- **真相**：宏观政策、行业格局、企业战略任何一项变化，都可能让原本的协整对子突然脱钩——中信 vs 华泰协整失效就是活教材。
- **避坑动作**：至少每季度重做一次 coint 检验，关系不显著立即清仓。实盘代码里加「每季度自动复检」的定时任务——不复检的策略迟早被现实教训。

## 坑 5：把季节效应当成趋势（伪非平稳）

- **现象**：电力消费、能源价格、农产品等序列在 ADF 上被误判「非平稳」。
- **真相**：有些序列有强季节性，看起来均值漂移，其实是周期重复。ADF 分不清「趋势漂移」和「季节循环」。
- **避坑动作**：先做 STL 分解（趋势 / 季节 / 残差），分别测各分量；或用 KPSS 检验交叉验证。金融数据季节性较弱，但月度效应、周内效应仍存在（比如 1 月效应、周一效应）。

## 实战 SOP：时间序列与协整

| # | 规则 |
| --- | --- |
| 01 | 任何时间序列分析必先 ADF 检验——不平稳就先差分（对数收益） |
| 02 | 金融价格几乎都不平稳 / 收益率几乎都平稳——默认入门动作就是转对数收益 |
| 03 | 配对交易必须 ADF（各自非平稳）+ coint（Engle-Granger 显著）两步验证，缺一不可 |
| 04 | spread 自身 ADF 平稳（p < 0.05）才能进入交易阶段 |
| 05 | 协整窗口最少 252 天 / 稳健 750–1250 天 / 必须含极端时段 |
| 06 | 实盘配对每季度复检协整关系，p > 0.05 立即清仓 |
| 07 | 配对资金占比 < 30%，避免单一关系崩溃带来灾难 |
