# Day 005 · 收益率的几种算法

> 看似简单、实则关键的一课：如果「算收益率」这一步就错了，后面所有的夏普比率、年化指标全都是错的。

- 📂 阶段：P1 · 量化基础
- ⏱️ 时长：约 18 分钟
- 🎯 关键词：简单收益、对数收益、年化、复权

## 这节课为什么重要

可以把量化学习想象成盖楼：**收益率计算就是地基**。地基不平，楼盖得再高也是歪的——你后面学的 Sharpe（夏普比率）、回撤、波动率，全都建立在「收益率算对了」这个前提上。

这节课还要揭穿一个很多人（包括不少基金经理）都犯过的小错误：**把月度收益直接乘以 12 当成年化收益**。2018 年就有私募因为这个错误，把真实年化约 30% 的业绩「美化」成了 50%，被监管核查发现。

> [!NOTE]
> **本节最重要的一句话：简单收益不可加，对数收益可加。**
> 这一句话决定了你今后做统计计算时该用哪一种收益率。

## 学习目标

听完这节课，你应该能做到：

- [x] 分清**简单收益**和**对数收益**的数学差别，知道各自该在什么场景用
- [x] 正确地做**年化**，并明白为什么是 √252 而不是 √365
- [x] 看懂为什么**对数收益**更适合量化建模和计算夏普比率
- [x] 会处理收益率的常见问题：停牌、缺失值、异常值、复权
- [x] 能用 Python 实现 5 种收益率算法，并跑出对比结果

## 一点历史八卦

- **1900 年**，法国数学家 Louis Bachelier 在博士论文《投机理论》里第一次用对数收益给股价建模——他比爱因斯坦还早 5 年用了同样的数学。可惜论文当年没人理，直到 60 年后被 Samuelson 重新发现，成了现代金融数学的源头。
- **1973 年**，著名的 Black-Scholes-Merton 期权定价公式发表，核心假设之一就是「对数收益服从正态分布」——这个假设影响了之后 50 年的量化金融。

> [!TIP]
> 不读历史的策略走不远。记住 Bachelier、Black-Scholes、Samuelson 这三个名字，以后会在很多地方反复遇到。
